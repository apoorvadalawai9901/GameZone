# GameZone

A simple Rock-Paper-Scissors browser game.

## Files

- `rock-paper-scissors.html` — Rock-Paper-Scissors game page
- `rock-paper-scissors.css` — Rock-Paper-Scissors styling
- `rock-paper-scissors.js` — Rock-Paper-Scissors logic

## How to play

1. Open `rock-paper-scissors.html` in a browser.
2. Click one of the buttons: Rock, Paper, or Scissors.
3. See the computer choice and the result.
4. Click Reset to start again.
